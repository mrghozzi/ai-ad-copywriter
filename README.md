# AI Ad Copywriter (Gemini AI)

A smart extension for the **MYADS** platform that integrates Google Gemini AI to help advertisers create professional and high-converting ad copy (titles and descriptions) with a single click.

## Features
- **Gemini AI Integration:** Uses the advanced **Gemini 3 Flash** engine for creative and lightning-fast content generation.
- **Premium Admin Interface:** Dedicated settings page in the admin panel to configure the API key and test the service.
- **Smart Assistant Integration:** Injected "Magic Wand" icon across all ad creation and promotion pages (Banners, Links, Smart Ads, Exchange).
- **Easy to Use:** Enter your topic, get 5 suggestions, and apply your favorite one instantly.
- **Full Multilingual Support:** Native support for both Arabic and English interfaces, with content generation adapting dynamically to your input.

## Installation & Usage
1. Activate the plugin from the Extensions section in the Admin Panel.
2. Navigate to **Admin Panel > AI Ad Assistant**.
3. Enter your **Gemini API Key** (get it from [Google AI Studio](https://aistudio.google.com/app/apikey)).
4. When creating a new ad, click the Magic Wand icon next to the Title or Description field.
5. Enter the ad topic, click "Generate", then choose the best suggestion.

## Requirements
- Valid Google Gemini API Key.
- Active internet connection for the server to communicate with the API.
