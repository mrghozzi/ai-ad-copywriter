# AI Ad Copywriter - Changelog

## v1.1.0
* **Core Update**: Migrated AI engine to the faster and more advanced **Gemini 3 Flash** (preview) to ensure continued compatibility after Google deprecated older versions.
* **Localization**: Added full multi-language support (English and Arabic) with dynamic UI switching based on the user's active language.
* **Enhancement**: Broadened the Smart Assistant injection to fully support the unified Promotions page (`/ads/promote`) and all its sub-tabs (Banners, Links, Exchange).
* **Fix**: Resolved an API versioning issue (`v1beta` vs `v1`) that caused generation failures.

## v1.0.0
* **Initial Release**: First version of the extension.
* **Feature**: Integrated Google Gemini AI engine.
* **UI**: Modern Premium admin interface for API configuration and generation testing.
* **Integration**: Added AI assistant functionality to Banner, Link, and Smart Ad creation pages.
* **UX**: Popup modal system for choosing suggestions and auto-filling form fields.
* **Branding**: Added professional AI-themed plugin thumbnail.
